Resources expose **readable data** to the model's context.

## Static Resource

```ts
import { Resource } from '@frontmcp/sdk';

@Resource({
  name: 'config',
  uri: 'app://config',
  mimeType: 'application/json',
})
class Config {
  execute() {
    return { version: '1.0', env: 'production' };
  }
}
```

**Function style:**

```ts
import { resource } from '@frontmcp/sdk';

const Config = resource({
  name: 'config',
  uri: 'app://config',
})(() => ({ version: '1.0' }));
```

## Resource Template

Dynamic URIs with parameters (RFC 6570):

```ts
import { ResourceTemplate } from '@frontmcp/sdk';

@ResourceTemplate({
  name: 'user-profile',
  uriTemplate: 'users://{userId}/profile',
})
class UserProfile {
  execute(uri: string, params: Record<string, string>) {
    return { id: params.userId, name: 'John' };
  }
}
```

**Function style:**

```ts
import { resourceTemplate } from '@frontmcp/sdk';

const UserProfile = resourceTemplate({
  name: 'user-profile',
  uriTemplate: 'users://{userId}/profile',
})((uri, params) => ({ id: params.userId }));
```

---

## Register

```ts
@App({
  resources: [Config, UserProfile],
})
class MyApp {}
```

## Return Values

```ts
// Object → auto-serialized to JSON
return { data: 'value' };

// String → text content
return 'plain text';

// Buffer → binary blob
return Buffer.from(data);

// Full control
return {
  contents: [
    { uri, mimeType: 'text/plain', text: 'content' },
    { uri, mimeType: 'image/png', blob: 'base64...' },
  ]
};
```

## Metadata

| Field | Required | Description |
|-------|----------|-------------|
| `name` | Yes | Identifier |
| `uri` | Yes* | Static URI (*Resource only) |
| `uriTemplate` | Yes* | RFC 6570 template (*ResourceTemplate only) |
| `title` | No | Display name |
| `description` | No | LLM hint |
| `mimeType` | No | Content type |
| `icons` | No | UI icons |

## Context

Class resources can access context via `this`:

```ts
@Resource({ name: 'data', uri: 'app://data' })
class Data {
  execute() {
    this.uri;        // requested URI
    this.params;     // template params
    this.authInfo;   // auth context
    this.get(Token); // resolve provider
    this.fetch(url); // built-in fetch
  }
}
```
