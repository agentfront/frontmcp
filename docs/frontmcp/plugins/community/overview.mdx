## Using Community Plugins

```bash
npm install @frontmcp/plugin-postgres
```

```typescript
import { FrontMcp } from '@frontmcp/sdk';
import { PostgresPlugin } from '@frontmcp/plugin-postgres';

@FrontMcp({
  plugins: [
    new PostgresPlugin({
      connectionString: process.env.DATABASE_URL
    })
  ]
})
export class MyServer {}
```

## Publishing Your Plugin

### 1. Create Plugin Package

```bash
mkdir my-plugin
cd my-plugin
npm init
```

### 2. Implement Plugin Interface

```typescript
import { Plugin } from '@frontmcp/sdk';

export class MyPlugin implements Plugin {
  name = 'my-plugin';
  // Implementation
}
```

### 3. Add Documentation

Include a README with:
- Installation instructions
- Configuration options
- Usage examples
- API reference

### 4. Publish

```bash
npm publish
```

### 5. Submit to Registry

Open a PR to add your plugin to the community registry:

```
https://github.com/agentfront/frontmcp-plugins
```

## Plugin Guidelines

### Quality Standards

- TypeScript types for all exports
- Comprehensive error handling
- Unit tests with >80% coverage
- No security vulnerabilities

### Documentation

- Clear installation instructions
- Configuration reference
- Usage examples
- Changelog

### Maintenance

- Respond to issues within 7 days
- Keep dependencies updated
- Follow semantic versioning
