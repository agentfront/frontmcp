## Install FrontMCP

FrontMCP targets TypeScript devs. Use your favorite package manager:

<CodeGroup>
    ```bash npm
    npm i @frontmcp/sdk
    ````

    ```bash pnpm
    pnpm add @frontmcp/sdk
    ```

    ```bash yarn
    yarn add @frontmcp/sdk
    ```

</CodeGroup>

Optional add-ons (install as needed):

* `@frontmcp/adapters` — e.g., OpenAPI tool loader
* `@frontmcp/plugins` — caching, extra hooks, etc.


<Info>
    FrontMCP is framework-style: your **decorated server class** boots the HTTP transport (Streamable HTTP) by default.
    No separate CLI required.
</Info>
