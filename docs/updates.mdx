<Update label="FrontMCP 0.3.0" description="November 2025" tags={["Releases"]}>
    <Card
        title="FrontMCP 0.3.0: Typed hooks, caching, unified CLI"
        href="https://github.com/agentfront/frontmcp/releases/tag/v0.3.0"
        cta="Read the release notes"
    >
        FrontMCP 0.3.0 unifies the runtime around a single SDK, adds typed flow hooks across tool and HTTP flows, and ships a built-in cache plugin plus hardened automation.

        #### Breaking changes
        - **Import from `@frontmcp/sdk`**: the deprecated `@frontmcp/core` package is removed; update any remaining imports to `@frontmcp/sdk`.

        #### Features
        - 99d **Typed flow hooks**: use `ToolHook`, `ListToolsHook`, and `HttpHook` helpers for Will/Stage/Did/Around hooks across each flow.
        - 9e0 **Cache plugin**: register `CachePlugin` for Redis or in-memory caching scoped by tool metadata.
        - ae0fe0f **frontmcp CLI**: run `frontmcp dev`, `build`, `init`, `doctor`, `inspector`, and `create` directly from the published CLI in this repo.

        #### Fixes
        - 4e1 **OpenAPI adapter**: JSON request bodies are stringified before fetch, preventing malformed payloads.

        #### Build & CI
        -  Hardened release workflows for branch creation, publishing, and Codex documentation updates.
    </Card>

</Update>
